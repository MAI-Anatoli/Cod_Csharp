## Эта команда создает новый проект
```sh
dotnet new console
```

## Эта команда запустит проект
```sh
dotnet run 
```

```sh
Write ();
```
## В конце перейти на новую строку это делает следующая команда
```sh
WriteLine();
```
## Считать строку из терминала может эта команда
```sh
ReadLine();
```
